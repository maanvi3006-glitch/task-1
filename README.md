# Task1_SQL_Analytics

This project is a professional PostgreSQL analytics assignment setup. Use `queries.sql` to store your SQL analysis queries, `INSIGHTS.md` for results-driven commentary, and `results/` to save exported CSV outputs.

## Files

- `queries.sql` — SQL query templates and final analytics queries
- `INSIGHTS.md` — key findings, recommendations, and business insights
- `README.md` — project overview and usage instructions
- `results/` — exported query results as CSV files

## How to use

1. Open this folder in Visual Studio Code.
2. Install PostgreSQL and the VS Code extensions described in the assignment guidance.
3. Use SQLTools with the PostgreSQL driver to connect to your local database.
4. Run queries from `queries.sql`, then export results into `results/`.
5. Record your observations and recommendations in `INSIGHTS.md`.

## Recommended query workflow

- Use `Ctrl + Shift + P` → `SQLTools: Add New Connection`
- Connect to `localhost:5432` with your PostgreSQL credentials
- Open `queries.sql`, select a query, and run it with `Ctrl + E Ctrl + E`
- Export the result grid to a CSV file in `results/`

## Submission

Compress this folder as `Task1_SQL_Analytics.zip` after you finish.
